# Contributing to Wine-Quality-Classification
Thanks for considering making contributions to this project!
The following content will guide you to make contribution for the project.

## Code Contributions
### Set up your environment
After clone thev repository, you can create your vitual environment by 'conda-lock.yml' file which included in the repository. (we support differnt opreation system)

### Creating a branch
we suggest you to create your own branch and do the changes in your own branch. (please do not try to directly push the changes to main branch).

### Testing the changes you made
It is highly recommanded that you test your changes to make sure it really works and have improvement for the project before you submit oyur changes to main branch.

### Creating your pull request
It is more helpful to add the reason why you do the changes and give some comments that why it is helpful for us to solve the problem, and the test case you made. And your PR will be response by our contributor and all the people who care about our project.

## Documentation Contributions
### Improving documentation clarity
Carefully read the documentation and identify any parts that confuse you or are easily misunderstood. Edit directly in your branch, then submit a pull request.

### Adding usage examples
Write and test new usage examples. Add examples to the appropriate section. Submit a pull request with the new content.

### Updating methodological process
provide some suggestion about new data processing methods, model designs, experimental procedures, or using diagrams to illustrate the principles. Then submit a pull request. (you do not need to upload your code, you can just share your thinking).

### Fixing typos and format problem
Correct typos, punctuation, layout, image links, and table formatting issues in various documents. Then,submit a pull request.

## Code of Conduct
Please read and respect our CODE_OF_CONDUCT.md and follow it to make a great community environment for everyone.
We really appreciate you for helping us improve this project!
